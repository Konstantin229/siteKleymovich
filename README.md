# Сайт с преднамеренными SEO-ошибками

## Описание проекта
Учебный проект для демонстрации распространённых SEO-ошибок на веб-сайтах. Сайт состоит из трёх страниц с разным уровнем SEO-оптимизации.

## Структура сайта
1. **index.html** - Главная страница с идеальным SEO
2. **about.html** - Страница "О компании" с преднамеренными ошибками
3. **contacts.html** - Страница "Контакты" с дополнительными ошибками

## Преднамеренные SEO-ошибки

### На странице about.html:
1. Слишком длинный тег `<title>` (более 60 символов)
2. Отсутствует тег `<meta name="description">`
3. Два тега `<h1>` на одной странице
4. Пустые атрибуты `alt` у изображений

### На странице contacts.html:
1. Неправильный порядок мета-тегов
2. Отсутствие aria-label у навигации
3. Изображения без указания размеров
4. Ссылки без атрибута title

## Как использовать

1. Клонируйте репозиторий
2. Загрузите файлы на GitHub
3. Включите GitHub Pages в настройках репозитория
4. Проверьте сайт по адресу: `https://ваш-логин.github.io/репозиторий/`

## Инструменты для проверки

1. **W3C Validator** - проверка HTML
2. **Google PageSpeed Insights** - скорость загрузки
3. **Яндекс.Вебмастер** - индексация сайта
4. **Google Search Console** - анализ SEO

## Лицензия
Учебный проект. Свободное использование с указанием авторства.
